## 쉼터
익명성을 보장하여 마음 편히 문제 혹은 고민을 털어놓는 커뮤니티 서비스  

## 주요기능
- 글 쓰기, 글 답변 , 글 확인 하기 기능
- 나의 화원 기능
- 오늘의 글, 책, 음악 추천 기능
- 회원관리 기능(로그인,회원가입,탈퇴)
- 익명성을 보장하는 커뮤니티 게시판
